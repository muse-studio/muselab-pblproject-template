\section{VOICEPEAKによる発話イントネーションの取得}\label{sec:huroku1}

本文\ref{sec:3.3}節で記した，発話イントネーションの取得方法について記す．\par
VOICEPEAKのイントネーションは，\figref{henkan_frow}の左図のように相対的な値となっている．そこで，\figref{henkan_frow}の右図のように，フレーズのイントネーションの全体の幅を取得することにより，イントネーションの割合を取得することができる．\par

\begin{figure}
\centering
\includegraphics[width=\columnwidth]{fig/henkan_frow.png}
\caption{イントネーションの取得方法イメージ図}
\label{henkan_frow}
\end{figure}

取得の概要は以上である．この概要に基づき，詳しい取得の順序を示す．詳しい取得の順序は下記の通りである．
\begin{musequote}
\begin{enumerate}
\renewcommand{\labelenumi}{\arabic{enumi})}
\item 発話イントネーションを取得したい語句で，VOICEPEAKのプロジェクトファイルを作成する
\item 作成したファイルをJSONファイルに変換する
\item イントネーションの調整パラメータを取得する
\item イントネーションの最高値と最低値を取得する
\item イントネーションの幅を調べる
\item 各イントネーションの値が幅に対して何\%にあるかを算出する
\item パーセントの最小値と最大値を0\%，100\%とし，その他の値の位置をパーセントで算出する
\end{enumerate}
\end{musequote}
この順序に則り，発話イントネーションを取得していく．\par
まず，VOICEPEAKで“あなたにあいたくて”“ひかりのほう”の合成音声のプロジェクトファイルを作成した．作成したプロジェクトファイルのイントネーションは\figref{intone_vp}のようになっている．\figref{intone_vp}のようなプロジェクトファイルを作成することにより，イントネーションの全体の幅を取得することができる．

作成したプロジェクトファイルをテキストエディタで開き，成形すると，終端にNull文字のあるJSONファイルとなる．実際に成形した後のコードの一部が\figref{json_seikei}である．\par
この図は“ひかりのほう”と言わせたときのJSONファイル“ひ”の部分で，各項目で合成音声の声音を制御している．イントネーションに関わる部分は，\figref{json_seikei}の“i”である．\figref{hikui}が\figref{intone_minus}の「ひ」（イントネーションの最低値），\figref{takai}が\figref{intone_plus}の「ひ」（イントネーションの最高値）となっている．\par
他の箇所に関しても同様に値を確認し，各単音の幅を調べた．すると，どの値においても6，または6にかなり近い値となったことが，本文の\figref{intone_henkan}の“イントネーションの幅”からもわかる．そのため，本研究では幅を6として扱う．

幅を6としたため，各イントネーションの絶対値および割合を求めていく．\par
各イントネーションの絶対値については，各イントネーションの最低値の絶対値であるため，最低値からマイナスの記号をとった値が絶対値となる．\par
絶対値を取得できたため，それが全体6からみてどの位置にあるかの割合，及び最低値と最高値を0\%，100\%としたときの割合を出していく．\par
結果が本文の\figref{intone_henkan}の“パーセント”及び“パーセント（真）”である．本研究では，「パーセント（真）」を使用した．
